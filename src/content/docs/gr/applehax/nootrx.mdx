## Προαπαιτούμενα

- Ένα OpenCore EFI διαμορφωμένο σύμφωνα με τον οδηγό μας.
- Να μην χρησιμοποιήτε το `WhateverGreen` ή το `NootedRed` διότι το `NootRX` δεν είναι συμβατό με εκείνες τις επεκτάσεις.
- Κάντε τις ενημερώσεις συστήματος του macOS (τις μικρές ενημερώσεις, δεν χρειάζεται να πάτε από το `macOS 11` στο `macOS 12` για παράδειγμα).

## Συχνές ερωτήσεις

<Aside type="danger">
Η λίστα αυτή δεν είναι πλήρης, για περισσότερες πληροφορίες θα πρέπει να ελέγξετε τις [αναφορές σφαλμάτων](https://github.com/ChefKissInc/NootRX/issues), ενδεχομένως και τις συζητήσεις.
</Aside>

### Βλέπω πράσινα και ροζ αλλοιώματα ενώ περιεχόμενο DRM αναπαράγεται στο παρασκήνιο από έναν περιηγητή με software DRM (αντί για hardware) στο Navi 22.

Δεν υπάρχει διαθέσιμη λύση, ακόμα.

### Ορισμένα 3D προγράμματα ή παιχνίδια εμφανίζουν μια πράσινη οθόνη ή πράσινα αντικείμενα στο Navi 22.

Δεν υπάρχει διαθέσιμη λύση, ακόμα. Ορισμένα προβλήματα μπορεί να ανακουφιστούν με την υποβάθμιση στα macOS 12.

### Περιεχόμενο τύπου FairPlay 1.x/4.x είτε εμφανίζει μια μαύρη οθόνη είτε προκαλεί κρασάρισμα στο Navi 22.

Δεν υπάρχει διαθέσιμη λύση, ακόμα.
