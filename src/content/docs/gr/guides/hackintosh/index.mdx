Καλώς ήρθατε στον Οδηγό Hackintosh ChefKiss.

Αυτός ο οδηγός θα σας συνοδεύσει στη διαδικασία εγκατάστασης του macOS στον υπολογιστή σας.

<Aside type="tip">
Ένα Hackintosh είναι ένας υπολογιστής που τρέχει το macOS σε ένα μη Apple σύστημα. Σούπερ!
</Aside>

## Προϋποθέσεις

<CardGrid>
    <Card icon="ph:clock-clockwise-duotone" title="Χρόνος και υπομονή">
    Η δημιουργία ενός Hackintosh είναι μια σχετικά χρονοβόρα διαδικασία. Δεν μπορούν να παραλειφθούν βήματα. Μην ξεκινήσετε να εργάζεστε σε αυτό εάν έχετε άλλα πράγματα να κάνετε. Τα Hackintoshes γενικά **δεν** είναι κάτι στο οποίο πρέπει να βασίζεστε ως εργαλείο εργασίας.
    </Card>
    <Card icon="ph:plugs-connected-duotone" title="Ένα συμβατό σύστημα">
    Η CPU, η GPU, η μητρική πλακέτα, η κάρτα Ethernet, η κάρτα Wi-Fi και άλλα είναι σημαντικά για το εάν το μηχάνημά σας μπορεί να τρέχει το macOS: Σε αντίθεση με το Windows και το Linux, το macOS δεν είναι συμβατό με κάθε υπολογιστή x86. Υπάρχει ακόμα μια σημαντική ποσότητα μη υποστηριζόμενου υλικού, παρόλο που υπάρχουν αρκετές προσπάθειες.
    </Card>
    <Card icon="ph:terminal-window-duotone" title="Βασικές γνώσεις γραμμής εντολών">
    Απαραίτητο για την εκτέλεση των βημάτων που περιέχονται στον οδηγό. Δεν μπορώ να σας βοηθήσω με το πώς να αλλάξετε τον φάκελο στην γραμμή εντολών.
    </Card>
    <Card icon="ph:usb-duotone" title="Μια αρκετά μεγάλη μονάδα USB">
    Χρειάζεστε τουλάχιστον 16GiB για να δημιουργήσετε μια USB μονάδα ανάκτησης εκτός σύνδεσης (μόνο στο macOS) και 2GiB εάν δημιουργείτε μια USB μονάδα ανάκτησης online (σε όλες τις πλατφόρμες).
    </Card>
    <Card icon="ph:network-duotone" title="Μια συμβατή κάρτα δικτύου">
    Εάν χρησιμοποιείτε online ανάκτηση. Όπως αναφέρθηκε προηγουμένως, το macOS δεν είναι συμβατό με όλο το υλικό, οπότε δεν μπορείτε απλώς να έχετε οποιαδήποτε κάρτα δικτύου.
    </Card>
    <Card icon="ph:computer-tower-duotone" title="Ένας λειτουργικός υπολογιστής">
    Χρειάζεστε κάτι για να εκτελέσετε όλες τις εντολές και τα εργαλεία για να δημιουργήσετε την USB μονάδα με όλα τα απαραίτητα αρχεία.
    </Card>
    <Card icon="ph:seal-check-duotone" title="Ενημερωμένο υλικολογισμικό">
    Το macOS είναι πολύ επιλεκτικό. Εάν το υλικό σας έχει οποιαδήποτε σημαντική "μη τυποποιημένη" ή "αναπάντεχη" συμπεριφορά ή σφάλμα, δεν θα λειτουργήσει.
    </Card>
</CardGrid>
