---
title: Έναρξη
description: Συγκεντρώστε τα αρχεία που απαιτούνται για ένα Hackintosh EFI.
---

Το πρώτο βήμα για να λειτουργήσει ένα Hackintosh είναι η συγκέντρωση των απαιτούμενων αρχείων. Αυτά τα αρχεία είναι απαραίτητα για την εκκίνηση του macOS και για την προσθήκη υποστήριξης σε διάφορες συσκευές.

Μην ξεχνάτε ότι η υπομονή είναι το κλειδί. Χωρίς τα κατάλληλα αρχεία, το macOS δεν θα εκκινήσει καθόλου, θα λείπει η υποστήριξη για τα περιφερειακά σας ή θα έχει τυχαία προβλήματα γενικά.

Έχουμε χωρίσει τη διαδικασία συγκέντρωσης αρχείων σε πολλά μέρη.

- [OpenCore](/gr/guides/hackintosh/gathering-files/opencore/), αυτό που εκκινεί το macOS
- [Εγκαταστάτης](/gr/guides/hackintosh/gathering-files/installer/), λήψη των επίσημων αρχείων εγκατάστασης του macOS από την Apple
- [ACPI](/gr/guides/hackintosh/gathering-files/acpi/), οι απαιτούμενες προσθήκες και διορθώσεις ACPI
- [Kexts](/gr/guides/hackintosh/gathering-files/kexts/), η ποικιλία των απαιτούμενων οδηγών
