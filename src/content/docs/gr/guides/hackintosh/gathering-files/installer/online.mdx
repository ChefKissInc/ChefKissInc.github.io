## Εκτέλεση της εντολής

Ανοίξτε ένα τερματικό (`cmd` στα Windows) και αλλάξτε τον κατάλογο σε `OpenCorePkg/Utilities/macrecovery`.

```bash
cd ~/Downloads/OCStuff
cd OpenCorePkg/Utilities/macrecovery
```

Τώρα, εκτελέστε την εντολή που αντιστοιχεί στην έκδοση του macOS που θέλετε να εγκαταστήσετε.

Αντιγράψτε και επικολλήστε την εντολή στην γραμμή εντολών που ανοίξατε προηγουμένως.

Για παράδειγμα, αν θέλετε να κατεβάσετε το macOS Ventura (13).

```bash
cd ~/Downloads/OCStuff
cd OpenCorePkg/Utilities/macrecovery
python3 macrecovery.py -b Mac-B4831CEBD52A0C4C -m 00000000000000000 download
```

<Aside type="caution">
Οι χρήστες των Windows μπορεί να χρειαστεί να χρησιμοποιήσουν `py -3` ή `python` αντί για `python3`.
</Aside>

## Έλεγχος των αρχείων

Μετά την ολοκλήρωση της εκτέλεσης του script, θα υπάρχει ένας νέος φάκελος με το όνομα `com.apple.recovery.boot` μέσα στον τρέχοντα κατάλογο του τερματικού (σε αυτή την περίπτωση, ο φάκελος `macrecovery`).

Μέσα σε αυτόν, θα πρέπει να υπάρχουν είτε:

- `BaseSystem.dmg` και `BaseSystem.chunklist`, ή
- `RecoveryImage.dmg` και `RecoveryImage.chunklist`.

Αν και τα δύο αρχεία σε οποιαδήποτε από τις δύο ομάδες είναι παρόντα, είμαστε εντάξει. Αντιγράψτε τον φάκελο `com.apple.recovery.boot` στον φάκελο `Drive` ή παρόμοια ονομασμένο
