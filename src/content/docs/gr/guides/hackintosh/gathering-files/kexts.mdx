Οδηγοί πυρήνα macOS. Το OpenCore μπορεί να εισάγει ή να αποκλείσει προγράμματα οδήγησης. Τα τοποθετείτε στο `EFI/OC/Kexts`, αλλά πρέπει επίσης να καθοριστούν στο `Config.plist`. Θα φροντίσουμε το τελευταίο σε ένα μεταγενέστερο βήμα.

<Aside type="tip" title="Το ήξερες;">
Το Kext σημαίνει Kernel EXTension (Επέκταση Πυρήνα). Γουάου!
</Aside>

<Aside type="note">
Στο macOS, τα kext εμφανίζονται ως ειδικός τύπος αρχείου με ένα λευκό κομμάτι Lego ως εικονίδιο.

Σε άλλα λειτουργικά συστήματα, τα kext εμφανίζονται ως απλοί φάκελοι που τελειώνουν σε `.kext`.
</Aside>

## Wi-Fi/Bluetooth

<CardGrid>
<Card title="Itlwm <code>10.13+</code>">
Υποστηρίζει τις περισσότερες κάρτες Wi-Fi Intel. Το `AirportItlwm` χρησιμοποιεί τη στοίβα Wi-Fi της macOS και το `Itlwm` χρησιμοποιεί το [`Heliport`](https://github.com/OpenIntelWireless/HeliPort) για σύνδεση σε δίκτυα.

<LinkButton icon="external" href="https://github.com/OpenIntelWireless/itlwm">Προβολή</LinkButton>
</Card>
<Card title="IntelBluetoothFirmware">
Υποστηρίζει τις περισσότερες κάρτες Wi-Fi Intel. Στο macOS 11 και χαμηλότερα, πρέπει να χρησιμοποιήσετε και τα 3 παρεχόμενα kexts.

<Aside type="caution">
Στο macOS 12+, πρέπει να χρησιμοποιήσετε `IntelBluetoothFirmware` και `IntelBTPatcher` από τα προαναφερθέντα, και `BlueToolFixup` από [`BrcmPatchRAM`](https://github.com/acidanthera/BrcmPatchRAM).
</Aside>

<LinkButton icon="external" href="https://github.com/OpenIntelWireless/IntelBluetoothFirmware">Προβολή</LinkButton>
</Card>
<Card title="AirportBrcmFixup">
Επιδιορθώνει τα kexts AirPort για να επιτρέψει σε μη-εγγενείς κάρτες Broadcom να χρησιμοποιήσουν την ενσωματωμένη στοίβα Wi-Fi.

<LinkButton icon="external" href="https://github.com/acidanthera/AirportBrcmFixup">Προβολή</LinkButton>
</Card>
<Card title="BrcmPatchRAM">
Επιδιορθώνει την στοίβα Bluetooth για να επιτρέψει σε μη-εγγενείς κάρτες Broadcom (και όλες τις άλλες στο macOS 12+ με `BlueToolFixup`) να χρησιμοποιήσουν την ενσωματωμένη στοίβα Bluetooth.

<LinkButton icon="external" href="https://github.com/acidanthera/BrcmPatchRAM">Προβολή</LinkButton>
</Card>
</CardGrid>

<Aside type="tip">
Εάν η κάρτα Wi-Fi δεν υποστηρίζεται και δεν έχετε Καλώδιο Δικτύου, μπορείτε να χρησιμοποιήσετε το [`HoRNDIS`](/Extras/Kexts/HoRNDIS.zip) για να ενεργοποιήσετε την σύνδεση USB μέσω Android. Η σύνδεση USB μέσω iPhone δεν λειτουργεί στα `recoveryOS`.
</Aside>
