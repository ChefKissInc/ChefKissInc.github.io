Ласкаво просимо до посібника з ChefKiss Hackintosh.

Цей посібник супроводжуватиме вас у процесі встановлення macOS на ваш ПК.

<Aside type="tip">
Hackintosh — це комп'ютер, на якому працює macOS на не-Apple системі. Неймовірно!
</Aside>

## Необхідні умови

<CardGrid>
    <Card icon="ph:clock-clockwise-duotone" title="Час та терпіння">
    Створення Hackintosh — це відносно тривалий процес. Жоден крок не можна пропустити. Не починайте працювати над цим, якщо у вас є інші справи. Hackintosh, як правило, __не__ є тим, на що слід покладатися як на робочу машину.
    </Card>
    <Card icon="ph:plugs-connected-duotone" title="Сумісна система">
    Ваш процесор, графічний процесор, материнська плата, мережева карта, карта Wi-Fi та інші компоненти важливі для того, чи зможе ваша машина запустити macOS: на відміну від Windows та Linux, macOS не сумісна з кожним x86 ПК; все ще існує значна кількість непідтримуваного обладнання, хоча й існують певні зусилля.
    </Card>
    <Card icon="ph:terminal-window-duotone" title="Базові знання командного рядка">
    Необхідно для виконання кроків посібника. Не можу допомогти вам з тим, як змінити робочий каталог у cmd.
    </Card>
    <Card icon="ph:usb-duotone" title="USB-накопичувач достатнього об'єму">
    Вам потрібно мінімум 16 ГБ для створення автономного відновлювального USB-накопичувача (тільки macOS) та 2 ГБ, якщо ви створюєте онлайн-відновлювальний USB-накопичувач (кросплатформний).
    </Card>
    <Card icon="ph:network-duotone" title="Сумісна мережева карта">
    Якщо ви використовуєте онлайн-відновлення. Як згадувалося раніше, macOS не сумісна з усім обладнанням, тому ви не можете використовувати будь-яку мережеву карту.
    </Card>
    <Card icon="ph:computer-tower-duotone" title="Працюючий комп'ютер">
    Вам потрібен щось для запуску всіх скриптів та інструментів, щоб створити USB-накопичувач з усіма необхідними файлами.
    </Card>
    <Card icon="ph:seal-check-duotone" title="Оновлена прошивка">
    MacOS дуже прискіплива; якщо ваше обладнання має будь-яку значну "нестандартну" або "неочікувану" поведінку або помилку, воно не працюватиме.
    </Card>
</CardGrid>
