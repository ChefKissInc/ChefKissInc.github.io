Драйвери ядра macOS. OpenCore може інжектити або блокувати драйвери. Ви розміщуєте їх у `EFI/OC/Kexts`, але їх також потрібно вказати в `Config.plist`. Ми подбаємо про останнє на пізнішому етапі.

<Aside type="tip" title="Чи знаєте ви?">
Kext означає Kernel EXTension. Вау!
</Aside>

<Aside type="note">
У macOS kexts відображаються як спеціальний тип файлу зі світло-білою цеглиною LEGO як піктограмою.

В інших операційних системах kexts відображаються як звичайні папки, які закінчуються на `.kext`.
</Aside>

<details>
<summary>Супутники</summary>

<Aside type="danger">
Ви не можете визначити тип пристрою лише за брендом або назвою на екрані.
</Aside>
<Aside type="danger">
Ви повинні використовувати лише один супутник, не ставте їх усі разом; вони можуть конфліктувати.
</Aside>

| Тип пристрою               | Супутник                                                                     |
| ------------------------- | ----------------------------------------------------------------------------- |
| HID                       | `VoodooI2CHID`                                                                |
| ELAN                      | `VoodooI2CELAN`                                                               |
| FTE                       | `VoodooI2CFTE`                                                                |
| Atmel Multitouch Protocol | `VoodooI2CAtmelMXT`                                                           |
| Synaptics                 | [`VoodooRMI`](https://github.com/VoodooSMBus/VoodooRMI) (також підтримує SMBus) |
| Alps                      | [`AlpsHID`](https://github.com/blankmac/AlpsHID)                              |

<Aside type="tip">
HID - дуже поширений тип пристрою. Якщо ви використовуєте ноутбук AMD, цілком ймовірно, що ваш тачпад підключений через нього. Проте, перевірте ще раз.
</Aside>
</details>

<LinkButton icon="external" href="https://github.com/VoodooI2C/VoodooI2C">Переглянути</LinkButton>
