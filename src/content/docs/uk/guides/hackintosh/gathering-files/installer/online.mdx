## Виконання команди

Відкрийте термінал (`cmd` у Windows) та змініть каталог на `OpenCorePkg/Utilities/macrecovery`.

```bash
cd ~/Downloads/OCStuff
cd OpenCorePkg/Utilities/macrecovery
```

Тепер запустіть команду, що відповідає версії macOS, яку ви хочете встановити.

Скопіюйте та вставте команду в термінал, який ви відкрили раніше.

Наприклад, якщо ви хочете завантажити `macOS Ventura (13)`.

```bash
cd ~/Downloads/OCStuff
cd OpenCorePkg/Utilities/macrecovery
python3 macrecovery.py -b Mac-B4831CEBD52A0C4C -m 00000000000000000 download
```

<Aside type="caution">
Користувачам Windows може знадобитися використовувати `py -3` або `python` замість `python3`.
</Aside>

## Перевірка файлів

Після завершення роботи скрипта з'явиться нова папка під назвою `com.apple.recovery.boot` у поточному каталозі терміналу (у цьому випадку, папка `macrecovery`).

Усередині неї має бути або:

- `BaseSystem.dmg` та `BaseSystem.chunklist`, або
- `RecoveryImage.dmg` та `RecoveryImage.chunklist`.

Якщо обидва файли в будь-якій з груп присутні, все добре; скопіюйте папку `com.apple.recovery.boot` до папки `Drive` або подібної.
