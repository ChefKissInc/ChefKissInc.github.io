---
title: Початок
description: Зберіть необхідні файли для EFI Hackintosh.
---

Перший крок до запуску Hackintosh – це збирання необхідних файлів. Ці файли потрібні для завантаження macOS та додавання до нього підтримки різних пристроїв.

Не забувайте, що терпіння – головне. Без належних файлів macOS взагалі не завантажиться, не матиме підтримки ваших периферійних пристроїв або матиме випадкові проблеми в цілому.

Ми розділили процес збирання файлів на кілька частин.

- [OpenCore](/uk/guides/hackintosh/gathering-files/opencore/), те, що фактично завантажує macOS
- [Installer](/uk/guides/hackintosh/gathering-files/installer/), завантаження офіційних файлів інсталятора macOS від Apple
- [ACPI](/uk/guides/hackintosh/gathering-files/acpi/), необхідні доповнення та патчі ACPI
- [Kexts](/uk/guides/hackintosh/gathering-files/kexts/), різноманітність необхідних драйверів
