## تشغيل الأمر

افتح المحطة الطرفية (`cmd` على نظام التشغيل Windows) وقم بتغيير الدليل إلى `OpenCorePkg/Utilities/macrecovery`.

```bash
cd ~/Downloads/OCStuff
cd OpenCorePkg/Utilities/macrecovery
```

الآن، قم بتشغيل الأمر المطابق لإصدار macOS الذي تريد تثبيته.

انسخ ولصق الأمر في المحطة التي فتحتها سابقًا.

على سبيل المثال، إذا كنت تريد تنزيل `macOS Ventura (13)`.

```bash
cd ~/Downloads/OCStuff
cd OpenCorePkg/Utilities/macrecovery
python3 macrecovery.py -b Mac-B4831CEBD52A0C4C -m 00000000000000000 download
```

<Aside type="caution">
قد يحتاج مستخدمو Windows إلى استخدام `py -3` أو `python` بدلاً من `python3`.
</Aside>

## التحقق من الملفات

بعد انتهاء تشغيل البرنامج النصي، سيكون هناك مجلد جديد يسمى `com.apple.recovery.boot` داخل الدليل الحالي للمحطة الطرفية (في هذه الحالة، مجلد `macrecovery`).

في داخله يجب أن يكون هناك إما:

- `BaseSystem.dmg` و `BaseSystem.chunklist`, أو
- `RecoveryImage.dmg` و `RecoveryImage.chunklist`.

إذا كان كلا الملفين في أي من المجموعتين موجودين، فهذا جيد؛ انسخ المجلد `com.apple.recovery.boot` إلى مجلد `Drive` أو مجلد ما شابه.
