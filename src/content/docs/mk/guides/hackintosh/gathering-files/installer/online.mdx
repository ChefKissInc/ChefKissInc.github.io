## Извршување на командата

Отворете терминал (`cmd` на Windows) и сменете го директориумот во `OpenCorePkg/Utilities/macrecovery`.

```bash
cd ~/Downloads/OCStuff
cd OpenCorePkg/Utilities/macrecovery
```

Сега, извршете ја командата што одговара на верзијата на macOS што сакате да ја инсталирате.

Копирајте ја и направете "paste" на командата на терминалот што претходно го отворивте.

Како, на пример, ако треба да преземете `macOS Ventura (13)`.

```bash
cd ~/Downloads/OCStuff
cd OpenCorePkg/Utilities/macrecovery
python3 macrecovery.py -b Mac-B4831CEBD52A0C4C -m 00000000000000000 download
```

<Aside type="caution">
Windows корисниците можеби ќе треба да користат „py -3“ или „python“ наместо „python3“.
</Aside>

## Проверка на датотеките

Откако скриптата ќе заврши со работа, ќе има нова папка наречена `com.apple.recovery.boot` во тековниот директориум на терминалот (во овој случај, папката `macrecovery`).

Во него, треба да има или:

- `BaseSystem.dmg` и `BaseSystem.chunklist`, или
- `RecoveryImage.dmg` и `RecoveryImage.chunklist`.

Ако и двете датотеки во која било група се присутни, ние сме добри; копирајте ја папката `com.apple.recovery.boot` во вашата папка `Drive` или слично.
