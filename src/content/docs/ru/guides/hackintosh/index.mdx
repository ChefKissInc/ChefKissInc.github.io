Добро пожаловать в руководство по Hackintosh от ChefKiss.

Это руководство поможет вам установить macOS на ваш ПК.

<Aside type="tip">
Hackintosh — это компьютер, на котором работает macOS на не-Apple системе. Вау!
</Aside>

## Необходимые условия

<CardGrid>
    <Card icon="ph:clock-clockwise-duotone" title="Время и терпение">
    Создание Hackintosh — относительно длительный процесс. Никакие шаги нельзя пропускать. Не начинайте работу над этим, если у вас есть другие дела. Hackintosh, как правило, __не__ следует использовать в качестве рабочей машины.
    </Card>
    <Card icon="ph:plugs-connected-duotone" title="Совместимая система">
    Ваш процессор, графический процессор, материнская плата, сетевая карта, карта Wi-Fi и другие компоненты важны для того, сможет ли ваша машина запустить macOS: в отличие от Windows и Linux, macOS не совместима с каждым ПК x86; всё ещё существует значительное количество неподдерживаемого оборудования, хотя предпринимаются различные попытки.
    </Card>
    <Card icon="ph:terminal-window-duotone" title="Базовые знания командной строки">
    Необходимо для выполнения шагов руководства. Не могу помочь вам с изменением рабочего каталога в cmd.
    </Card>
    <Card icon="ph:usb-duotone" title="USB-накопитель достаточного объёма">
    Вам потребуется минимум 16 ГБ для создания автономного загрузочного USB-накопителя (только macOS) и 2 ГБ, если вы создаёте онлайн загрузочный USB-накопитель (кроссплатформенный).
    </Card>
    <Card icon="ph:network-duotone" title="Совместимая сетевая карта">
    Если вы используете онлайн-восстановление. Как уже упоминалось, macOS не совместима со всем оборудованием, поэтому у вас не может быть просто любой сетевой карты.
    </Card>
    <Card icon="ph:computer-tower-duotone" title="Рабочий компьютер">
    Вам нужен компьютер для запуска всех скриптов и инструментов, необходимых для создания USB-накопителя со всеми необходимыми файлами.
    </Card>
    <Card icon="ph:seal-check-duotone" title="Обновлённое прошивки">
    macOS очень придирчива; если ваше оборудование имеет какое-либо значительное "нестандартное" или "неожиданное" поведение или ошибку, оно работать не будет.
    </Card>
</CardGrid>
