description: Сбор необходимых файлов для EFI Hackintosh.

Первый шаг к запуску Hackintosh — сбор необходимых файлов. Эти файлы необходимы для загрузки macOS и добавления поддержки различных устройств.

Не забывайте, что терпение — ключ к успеху. Без необходимых файлов macOS вообще не загрузится, будет отсутствовать поддержка периферийных устройств или будут возникать случайные проблемы.

Мы разделили процесс сбора файлов на несколько частей.

- [OpenCore](/ru/guides/hackintosh/gathering-files/opencore/), то, что фактически загружает macOS
- [Installer](/ru/guides/hackintosh/gathering-files/installer/), загрузка официальных файлов установщика macOS от Apple
- [ACPI](/ru/guides/hackintosh/gathering-files/acpi/), необходимые дополнения и патчи ACPI
- [Kexts](/ru/guides/hackintosh/gathering-files/kexts/), различные необходимые драйверы
