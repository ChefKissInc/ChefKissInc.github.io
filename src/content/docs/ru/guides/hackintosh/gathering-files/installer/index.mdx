Существует два типа установочных файлов macOS

- [Онлайн](/ru/guides/hackintosh/gathering-files/installer/online/), этот метод использует базовую среду восстановления macOS, называемую `recoveryOS`. Он загружает файлы ОС с серверов Apple во время установки, что требует стабильного подключения к интернету. Этот метод может быть выполнен практически на любой ОС.
- Оффлайн <Badge text="TODO" variant="note"/>, этот метод загружает полный установочный файл macOS, который не требует подключения к интернету во время установки.
