## Выполнение команды

Откройте терминал (`cmd` в Windows) и перейдите в директорию `OpenCorePkg/Utilities/macrecovery`.

```bash
cd ~/Downloads/OCStuff
cd OpenCorePkg/Utilities/macrecovery
```

Теперь выполните команду, соответствующую версии macOS, которую вы хотите установить.

Скопируйте и вставьте команду в ранее открытый терминал.

Например, если вы хотите загрузить `macOS Ventura (13)`.

```bash
cd ~/Downloads/OCStuff
cd OpenCorePkg/Utilities/macrecovery
python3 macrecovery.py -b Mac-B4831CEBD52A0C4C -m 00000000000000000 download
```

<Aside type="caution">
Пользователям Windows может потребоваться использовать `py -3` или `python` вместо `python3`.
</Aside>

## Проверка файлов

После завершения работы скрипта в текущей директории терминала (в данном случае, папке `macrecovery`) появится новая папка `com.apple.recovery.boot`.

Внутри неё должны находиться:

- `BaseSystem.dmg` и `BaseSystem.chunklist`, или
- `RecoveryImage.dmg` и `RecoveryImage.chunklist`.

Если оба файла из любой из групп присутствуют, всё в порядке; скопируйте папку `com.apple.recovery.boot` в папку `Drive` или аналогичную.
