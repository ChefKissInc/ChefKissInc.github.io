Драйверы ядра macOS. OpenCore может вводить или блокировать драйверы. Вы помещаете их в `EFI/OC/Kexts`, но их также необходимо указать в `Config.plist`. Мы позаботимся об этом на более позднем этапе.

<Aside type="tip" title="Знаете ли вы?">
Kext означает Kernel EXTension. Вау!
</Aside>

<Aside type="note">
В macOS kexts отображаются как специальный тип файла со значком в виде белоснежного лего-кирпичика.

В других операционных системах kexts отображаются как обычные папки, заканчивающиеся на `.kext`.
</Aside>

## Ввод

<CardGrid>
<Card title="VoodooPS2 <code>10.11+</code>">
Драйвер для периферийных устройств PS/2. Используется для клавиатур ноутбуков или мышей/клавиатур PS/2 на настольных компьютерах.

<LinkButton icon="external" href="https://github.com/acidanthera/VoodooPS2">Просмотр</LinkButton>
</Card>
<Card title="VoodooI2C <code>10.13+</code>">
Драйвер для устройств ввода I2C.

<details>
<summary>Спутники</summary>

<Aside type="danger">
Вы не можете определить тип устройства только по бренду или отображаемому имени.
</Aside>
<Aside type="danger">
Вы должны использовать только один спутник, не объединяйте их все вместе; они могут конфликтовать.
</Aside>

| Тип устройства            | Спутник                                                                            |
| ------------------------- | ---------------------------------------------------------------------------------- |
| HID                       | `VoodooI2CHID`                                                                     |
| ELAN                      | `VoodooI2CELAN`                                                                    |
| FTE                       | `VoodooI2CFTE`                                                                     |
| Atmel Multitouch Protocol | `VoodooI2CAtmelMXT`                                                                |
| Synaptics                 | [`VoodooRMI`](https://github.com/VoodooSMBus/VoodooRMI) (также поддерживает SMBus) |
| Alps                      | [`AlpsHID`](https://github.com/blankmac/AlpsHID)                                   |

<Aside type="tip">
HID — очень распространенный тип устройства. Если вы используете ноутбук AMD, очень вероятно, что ваш тачпад подключен через него. Тем не менее, проверьте еще раз.
</Aside>
</details>

<LinkButton icon="external" href="https://github.com/VoodooI2C/VoodooI2C">Просмотр</LinkButton>
</Card>
