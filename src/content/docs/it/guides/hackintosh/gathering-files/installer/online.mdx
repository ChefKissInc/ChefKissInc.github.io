## Eseguire il comando

Apri un terminale (`cmd` su Windows) e cambia la directory a `OpenCorePkg/Utilities/macrecovery`.

```bash
cd ~/Downloads/OCStuff
cd OpenCorePkg/Utilities/macrecovery
```

Ora, esegui il comando corrispondente alla versione di macOS che vuoi installare.

Copia e incolla il comando nel terminale che hai precedentemente aperto.

Come, ad esempio, se volessi scaricare `macOS Ventura (13)`.

```bash
cd ~/Downloads/OCStuff
cd OpenCorePkg/Utilities/macrecovery
python3 macrecovery.py -b Mac-B4831CEBD52A0C4C -m 00000000000000000 download
```

<Aside type="caution">
Gli utenti Windows potrebbero dover usare `py -3` o `python` invece di `python3`.
</Aside>

## Controllare i file

Dopo che lo script termina l'esecuzione, ci sarà una nuova cartella di nome `com.apple.recovery.boot` nella cartella corrente del terminale (in questo caso, la cartella `macrecovery`).

Dentro di essa, dovrebbero esserci:

- `BaseSystem.dmg` e `BaseSystem.chunklist`, o
- `RecoveryImage.dmg` e `RecoveryImage.chunklist`.

Se sono presenti entrambi i file di uno dei due gruppi, siamo a posto; copia la cartella `com.apple.recovery.boot` nella tua cartella `Drive` o simile.
