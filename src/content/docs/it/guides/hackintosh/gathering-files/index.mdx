description: Ottieni i file necessari per la EFI di un Hackintosh.
---

Il primo passaggio per far funzionare un Hackintosh è ottenere i file necessari. Questi file sono richiesti per l'avvio di macOS e per aggiungere a esso il supporto di varie periferiche.

Non dimenticare che la pazienza è cruciale. Senza i file corretti, macOS semplicemente non si avvierà, non avrà supporto per le tue periferiche, o avrà problemi casuali in generale.

Abbiamo diviso il processo di ottenimento dei file in vari passaggi.

- [OpenCore](/it/guides/hackintosh/gathering-files/opencore/), che avvia macOS
- [Installer](/it/guides/hackintosh/gathering-files/installer/), lo scaricamento dei file dell'installer di macOS ufficiali Apple
- [ACPI](/it/guides/hackintosh/gathering-files/acpi/), le aggiunte e patch ACPI necessarie
- [Kext](/it/guides/hackintosh/gathering-files/kexts/), i vari driver necessari
