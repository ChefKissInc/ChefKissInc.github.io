# ChefKissInc.github.io

The official ChefKiss site. The guide is currently a WIP.

To install dependencies, run

```sh
pnpm i
```

To run a dev server, run

```sh
pnpm dev
```

To build the site statically, run

```sh
pnpm generate
```
